How Accurate Is Shutter Count?
Can It Be Faked?

The shutter count embedded in a RAW file is highly accurate — but only if the file is original and unedited. Here's what you need to know before trusting a count when buying a used camera.

Two Places the Shutter Count Lives

Understanding shutter count accuracy requires knowing there are two distinct places the counter exists — with very different security properties:

1. The Internal Hardware Counter

Inside every camera is a non-volatile hardware counter that increments by 1 every time the mechanical shutter fires. This counter:

  • Is stored in protected firmware memory on the camera's main board.
  • Cannot be modified by editing image files — it's completely separate from file data.
  • Persists through firmware updates, battery removal, power cycles, and factory resets.
  • Can only be reset by an authorised service centre after physically replacing the shutter mechanism.

2. The EXIF Metadata in Each RAW File

When a photo is taken, the camera copies the current hardware counter value into the RAW file's EXIF MakerNote section. This copy:

  • Accurately reflects the hardware counter at the moment of capture.
  • Can be modified using EXIF editing software — the copy in the file is just data.
  • Is what tools like shuttercount.app, ExifTool, and similar read.
  • Is reliable only if the file is original and unedited.
Key distinction: The hardware counter cannot be faked. The file's EXIF copy can be edited. Always read shutter count from an original, unedited RAW file straight from the camera's memory card.

How Accurate Is Shutter Count from an Original RAW?

From a genuine, unedited original RAW file, the shutter count reading is exact — not an estimate, not a range. It's the precise value the camera's hardware recorded at the moment of capture.

For supported camera models (Canon, Nikon, Sony, Fujifilm and others), shuttercount.app reads the correct MakerNote tag for that specific model and returns the exact mechanical actuation count. There is no margin of error in the reading itself — only the question of whether the source file is trustworthy.

Can Shutter Count Be Faked?

Yes — with important limitations. A dishonest seller can edit the EXIF metadata in a RAW file to show a lower shutter count. Here's how that attack works and why it's detectable:

What a Seller Can Fake

  • Edit the MakerNote shutter count field in a specific RAW file using tools like ExifTool.
  • Send you that modified file, which will show a falsified count.
  • Share a screenshot of a low count (which proves nothing at all).
  • Share a JPEG export or converted DNG from editing software (which may strip or modify MakerNote data).

What a Seller Cannot Fake

  • The internal hardware counter in the camera — this can only be reset by authorised service with physical shutter replacement.
  • A fresh RAW file taken live in front of you (in person, or verified via video call) from the camera's memory card without any post-processing.
  • The consistency of shutter count across multiple files from different shooting sessions — a manipulated file will typically be inconsistent with surrounding files if you examine a batch.
The one rule that protects you: Request a freshly-taken RAW file directly from the camera's memory card, transferred to you without any editing software involvement. Drop it into shuttercount.app. If the seller refuses or cannot provide this, treat the count as unverified.

Red Flags When Buying a Used Camera

Be alert to these warning signs that a shutter count claim may not be trustworthy:

Red Flag Why It's Suspicious What to Do
Seller provides a screenshot Screenshots can be photoshopped or taken from any file Reject — ask for the actual RAW file
File was exported from Lightroom or Photoshop Export may strip or modify MakerNote data Ask for the original camera RAW, not an export
Suspiciously round shutter count (e.g. exactly 1,000) Natural counts are irregular; round numbers may indicate editing Ask for multiple files to cross-check
Count inconsistent with camera age and claimed use A 5-year-old "professional" camera with 2,000 actuations is unusual Ask for more files; request in-person or video verification
Seller refuses to provide a fresh RAW file Legitimate sellers have nothing to hide Major red flag — consider walking away
File metadata shows editing software in the MakerNote Any processing software could have modified data Cross-reference with other file metadata fields

How to Get a Trustworthy Shutter Count Reading

Follow these steps whether you're verifying your own camera or checking a used purchase:

  1. Take a photo with the camera set to mechanical shutter mode (disable silent/electronic).
  2. Remove the memory card and copy the RAW file directly to your computer — do not import via Lightroom, Photos, or any application that might process the file.
  3. Drop the original RAW file into shuttercount.app. The file never leaves your computer — it's parsed locally in your browser.
  4. The exact shutter count from the hardware counter (as recorded at the moment of capture) is displayed.

For buying a used camera remotely: ask the seller to perform steps 1–2 on camera and send you the raw file directly from the card — ideally over a shared folder or transfer service where they upload from the card, not from their Lightroom catalogue.

Best practice: When meeting in person to buy a used camera, bring a laptop, insert the seller's memory card yourself, copy a RAW file directly, and verify the count on the spot with shuttercount.app. This is the most reliable verification method available.

Why Different Tools Sometimes Show Different Counts

If you've read the shutter count with one tool and then tried another and got a different number, there are a few possible explanations:

  • Different tags being read: Some cameras store multiple counters in the MakerNote — e.g., total exposures (including electronic shutter) vs. mechanical-only. Different tools may read different tags.
  • Model-specific offset differences: Parsing the proprietary MakerNote is not officially documented by manufacturers. Reverse-engineered offsets can vary between tools, especially for newer camera models.
  • Newer camera not yet supported: If a camera model is very recent, some tools may not yet have the correct offset. ShutterCount.app is updated regularly to support new models.

The mechanical shutter actuation counter — which is what matters for assessing shutter wear — is the number reported by shuttercount.app.
