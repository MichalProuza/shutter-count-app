How to Check Shutter Count on the Fujifilm GFX 100RF

  1. Take any photo with your Fujifilm GFX 100RF and locate the .RAF file on your memory card or computer.
  2. Open shuttercount.app in any modern browser.
  3. Drag the RAF file onto the drop zone, or click to open a file picker.
  4. The shutter count and remaining life percentage appear instantly. No upload, no account required.
Card format caveat: Fujifilm stores the shutter count in the RAF file via MakerNote tag 0x1438 (ImageCount). This counter is tied to the memory card and resets to zero when the card is formatted. When buying a used GFX 100RF, ask for a RAW file from a card that has not been formatted since purchase, or request evidence of the count through another means (e.g., camera service menu).

How Does ShutterCount Read the GFX 100RF Shutter Count?

Like all Fujifilm X-series and GFX cameras, the GFX 100RF stores an image count in MakerNote tag 0x1438 in every RAF file. ShutterCount reads this tag directly in your browser without any upload.

The key limitation to be aware of: this counter is card-based, not body-based. Formatting the memory card resets the RAF ImageCount to zero, even though the physical shutter mechanism has no memory of this reset. When verifying a used camera's history, always check from an unformatted original card.

Fujifilm GFX 100RF Shutter Count — FAQ

How do I check the shutter count of my Fujifilm GFX 100RF?

Shoot a RAF RAW file with the mechanical shutter, then drop it into shuttercount.app. The count is read from MakerNote tag 0x1438 in your browser. Electronic shutter shots also increment this counter.

What is the rated shutter life of the Fujifilm GFX 100RF?

Fujifilm does not publish an official rating. Based on the broader GFX lineup, approximately 150,000 actuations is the conventional estimate for the mechanical shutter.

Does the electronic shutter affect the count?

The electronic shutter on the GFX 100RF bypasses the mechanical curtain and adds no mechanical wear. However, the RAF ImageCount tag 0x1438 may still increment for electronic shutter shots depending on firmware — always evaluate the count in context of how the camera was primarily used.

Can the shutter count on a GFX 100RF be reset or faked?

Formatting the memory card resets the in-file counter to zero. This is not the same as a body reset. A used GFX 100RF with a very low RAF counter may simply have had its card formatted. Only a Fujifilm authorised service centre can comment on the actual mechanical shutter actuation history from the body's internal logs.
