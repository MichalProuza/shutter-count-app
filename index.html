Podporované formáty a zdroje dat

Canon CR3 (R5, R6, R7, R8, R10…) — ISOBMFF kontejner, cesta: CTMD rec.8 → TIFF tag 0x000D. ✅ Potvrzeno: R6II/R8/R50 → 0x0D29 (uint32), R5/R6/R3 → 0x0AF1 (uint32), R6 Mark III → 0x086D (uint16, ImageCount). ⚠️ Experimentální: EOS RP/R/R7/R10 → 0x09C0.

Canon CR2 — ShutterCount je v EXIF pouze u profesionálních těl řady 1D/1Ds. Canon ho pro běžná těla (5D, 7D, 60D, 70D, 80D, Rebel/xxxD…) do souboru vůbec nezapisuje — jde o hardwarový čítač čitelný pouze přes USB PTP z kamery. Pro 1D/1Ds: MakerNote tag 0x93 (FileInfo). Pro 1DmkIII: tag 0x0d (CameraInfo[0x176]).

Nikon NEF — MakerNote tag 0x00A7 (ShutterCount). ✅ Všechna moderní těla.

Sony ARW / DNG — ARW: tag 0x9050 šifrovaný (b³ % 249). DNG: MakerNote v DNGPrivateData (Adobe MakN wrapper) s korekcí offsetů. Po dešifrování: offset 0x003a (A7III/IV, A9, A6400…), 0x0032 (A7I/II, SLT), 0x000a (A6700, A7CM2/CR).

Fujifilm RAF — RAF obsahuje embedded JPEG s plným EXIF. ShutterCount je v Fujifilm MakerNote: tag 0x1402 (ShotCount). ✅ Potvrzeno pro X-T3/4, X100V/VI, X-S10/20, GFX sérii. Starší těla (X-T1/2, X-Pro1/2) mohou tag neobsahovat.

Olympus / OM System ORF — MakerNote Equipment sub-IFD, tag 0x0101 (ShutterCount). ✅ Moderní Olympus E-system, OM-D, OM System (OM-1, OM-5). Starší Olymp\0 formát také podporován.
