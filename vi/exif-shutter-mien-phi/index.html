EXIF Là Gì Và Tại Sao Lưu Shutter Count?

EXIF (Exchangeable Image File Format) là chuẩn metadata được nhúng vào file ảnh. Ngoài thông tin phơi sáng (ISO, tốc độ màn trập, khẩu độ), EXIF còn chứa một phần riêng gọi là MakerNote — block dữ liệu đặc thù của từng nhà sản xuất.

Mỗi khi máy ảnh ghi file, nó cập nhật giá trị shutter count hiện tại vào MakerNote. Đây là cách dữ liệu được lưu trong từng hãng:

HãngĐịnh DạngVị Trí Trong EXIF
CanonCR3, CR2CameraInfo block — offset đặc thù theo model
NikonNEFTag ShutterCount trong Nikon MakerNote
SonyARWSony MakerNote — tag 0x9999 hoặc tương đương
FujifilmRAFFujifilm MakerNote — tag FrameNumber
Olympus/OM SystemORFKhông phải tất cả model đều lưu trong file
PanasonicRW2Thường không lưu trong file — kiểm tra qua menu

Tại Sao Dùng ShutterCount Thay Vì Phần Mềm Khác?

ShutterCount (shuttercount.app)

  • Miễn phí hoàn toàn — không tài khoản, không phí ẩn.
  • Không tải file lên server — file xử lý ngay trong trình duyệt, không rời khỏi máy tính bạn.
  • Dễ sử dụng — chỉ kéo thả file, kết quả ngay lập tức.
  • Hỗ trợ 210+ model — Canon, Nikon, Sony, Fujifilm, Olympus, Panasonic, DNG.
  • Hiển thị % vòng đời — tự động tính phần trăm màn trập đã dùng.

ExifTool (phần mềm dòng lệnh)

ExifTool là công cụ mạnh mẽ và miễn phí để đọc toàn bộ metadata EXIF. Cần cài đặt và dùng dòng lệnh:

exiftool -ShutterCount ten_file.NEF
exiftool -ShutterCount ten_file.CR3

Phù hợp với người dùng kỹ thuật muốn trích xuất toàn bộ metadata hoặc xử lý hàng loạt file.

Gợi ý: Đối với người dùng thông thường, ShutterCount là lựa chọn tốt nhất — nhanh, miễn phí và không cần cài gì.

Dữ Liệu EXIF Có Thể Bị Làm Giả Không?

Có và không. Phần mềm chỉnh sửa EXIF (như ExifTool) có thể thay đổi giá trị shutter count được nhúng trong file JPEG hoặc RAW đơn lẻ. Tuy nhiên:

  • Bộ đếm phần cứng trong máy ảnh không thể thay đổi bằng phần mềm — chỉ reset được sau khi thay màn trập tại trung tâm bảo hành chính hãng.
  • File bạn tự chụp tại chỗ khi kiểm tra máy sẽ phản ánh giá trị thật từ bộ đếm phần cứng.
  • File do người bán cung cấp có thể đã bị chỉnh sửa.
Lời khuyên bảo mật: Khi mua máy ảnh cũ, luôn chụp một file RAW bằng máy đó ngay tại chỗ gặp mặt và kiểm tra shutter count từ file đó — đừng tin vào file người bán đã gửi trước.

JPEG vs. RAW — Định Dạng Nào Tốt Hơn Để Kiểm Tra?

RAW luôn là lựa chọn tốt hơn để kiểm tra shutter count:

  • RAW (CR3, NEF, ARW, RAF) — luôn chứa shutter count đầy đủ trong MakerNote.
  • JPEG — nhiều model không nhúng shutter count vào JPEG, hoặc chỉ nhúng một phần thông tin. Một số model entry-level chỉ lưu trong RAW.

Nếu máy ảnh hỗ trợ RAW, hãy chuyển sang chế độ RAW, chụp một ảnh, rồi dùng file đó để kiểm tra. Kết quả sẽ chính xác nhất.
