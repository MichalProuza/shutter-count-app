EXIF MakerNote ಎಂದರೇನು?

EXIF (Exchangeable Image File Format) ಡೇಟಾ ಪ್ರತಿ ಫೋಟೋ file ನಲ್ಲಿ ಇರುತ್ತದೆ. ಇದರಲ್ಲಿ ಕ್ಯಾಮೆರಾ model, ISO, aperture, shutter speed ಮತ್ತು ಶಟರ್ ಕೌಂಟ್ ಸೇರಿದಂತೆ ಅನೇಕ ವಿವರಗಳಿರುತ್ತವೆ.

MakerNote ಭಾಗದಲ್ಲಿ ಪ್ರತಿ brand ತನ್ನದೇ ಆದ ರೀತಿಯಲ್ಲಿ ಶಟರ್ ಕೌಂಟ್ ಸಂಗ್ರಹಿಸುತ್ತದೆ. ShutterCount 210+ ಕ್ಯಾಮೆರಾ models ಗೆ ಈ MakerNote format ಅರ್ಥ ಮಾಡಿಕೊಂಡು ಶಟರ್ ಕೌಂಟ್ ತೋರಿಸುತ್ತದೆ.

ಗಮನಿಸಿ: ಶಟರ್ ಕೌಂಟ್ standard EXIF ಟ್ಯಾಗ್ ಅಲ್ಲ — ಇದು ಪ್ರತಿ manufacturer ನ ಖಾಸಗಿ MakerNote ನಲ್ಲಿ ಇರುತ್ತದೆ. ಆದ್ದರಿಂದ ಸಾಮಾನ್ಯ EXIF viewers ಇದನ್ನು ತೋರಿಸುವುದಿಲ್ಲ.

EXIF data ಏಕೆ ಕಾಣುತ್ತಿಲ್ಲ?

  • WhatsApp / Instagram — ಈ apps ಕಳಿಸುವಾಗ EXIF data ತೆಗೆದುಹಾಕುತ್ತವೆ
  • Screenshot — Screenshot ನಲ್ಲಿ original EXIF ಇರುವುದಿಲ್ಲ
  • Email compression — ಕೆಲವು email services ಫೋಟೋ compress ಮಾಡಿ EXIF ಕಳೆಯುತ್ತವೆ
  • Photo editing software — ಕೆಲವು editors save ಮಾಡುವಾಗ MakerNote ತೆಗೆದುಹಾಕುತ್ತವೆ
ಪರಿಹಾರ: ಯಾವಾಗಲೂ ಕ್ಯಾಮೆರಾ ಅಥವಾ memory card ನಿಂದ ನೇರವಾಗಿ original RAW ಅಥವಾ JPEG file ಬಳಸಿ.
