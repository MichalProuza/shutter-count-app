RAW File EXIF Data Explained:
Where Camera Metadata Is Stored

Every RAW file carries a rich set of metadata beyond the image itself — shutter speed, lens, GPS, and buried inside a proprietary block called the MakerNote, your camera's shutter count.

What Is EXIF Data?

EXIF stands for Exchangeable Image File Format. It's a standard for embedding metadata inside image files — established in the 1990s and now used by virtually every digital camera and smartphone on the market.

When your camera takes a photo, it writes two things into the file: the image data itself, and a block of structured metadata describing the capture conditions. This metadata block is the EXIF data.

EXIF data is why you can open a photo years later and still know exactly when it was shot, at what focal length, which aperture, and with which camera body. It travels with the file automatically — no input required from the photographer.

Key point: EXIF metadata is not just about exposure settings. Modern cameras embed dozens of fields — including proprietary data that reveals the camera's total shutter actuation count.

Standard EXIF Fields

The EXIF standard defines a common set of fields that all cameras use consistently. These are readable by any image viewer or EXIF editor:

Make / Model Camera brand and model name Canon EOS R6 Mark II

ExposureTime Shutter speed used 1/500 s

The MakerNote: Where Shutter Count Is Hidden

The standard EXIF fields above cover general photographic information. But each camera manufacturer also needs to store data that doesn't fit the standard — things like focus point positions, image stabilisation data, in-camera processing settings, and critically: the mechanical shutter actuation counter.

This is stored in a block called the MakerNote (EXIF tag 0x927C). It's a private, manufacturer-controlled section inside the EXIF data where each brand writes whatever proprietary information they choose, in whatever format they choose.

How Each Brand Stores Shutter Count

Brand RAW Format MakerNote Type Shutter Count Location
Canon CR2, CR3 Canon IFD sub-block Tag 0x0001 entry [2] (ShutterCount)
Nikon NEF, NRW Encrypted Nikon sub-IFD ShutterCount tag in Nikon MakerNote
Sony ARW, SR2 Sony IFD chain Tag 0x9999 / model-specific offset
Fujifilm RAF Fujifilm proprietary block RAF header + EXIF sub-block
Important: MakerNote formats are not publicly documented by manufacturers. The offsets and encoding are reverse-engineered by the photography community and tools like ExifTool. ShutterCount.app maintains up-to-date parsing for all supported camera models.

RAW Files vs. JPEG: EXIF Differences

Both RAW and JPEG files can contain EXIF data, but there are important differences — especially when it comes to shutter count.

RAW Files

  • Contain the full, unmodified MakerNote written directly by the camera firmware.
  • Shutter count is reliably present in supported camera models.
  • No in-camera processing that could affect metadata.
  • Larger files (20–80 MB depending on model and sensor).

JPEG Files

  • May have reduced MakerNote data — some cameras strip or truncate proprietary fields during JPEG compression.
  • Shutter count may or may not be present depending on camera model and firmware.
  • If the JPEG was exported from editing software (Lightroom, Photoshop, Capture One), the MakerNote may have been removed entirely.
  • Always prefer an original RAW file straight from the camera card for reliable shutter count extraction.
Best practice: For shutter count verification — especially when evaluating a used camera — ask for an original, unedited RAW file. Do not accept a screenshot, a JPEG export, or a file that has passed through editing software.

How to Read EXIF Data from a RAW File

There are several ways to inspect EXIF metadata:

Using ShutterCount.app (Recommended for Shutter Count)

  1. Take a photo in mechanical shutter mode (not silent/electronic).
  2. Transfer the RAW file to your computer — do not edit or convert it.
  3. Drop the file into shuttercount.app. No upload — the file is parsed entirely in your browser.
  4. The shutter count is extracted from the MakerNote and displayed instantly.

Using ExifTool (Full EXIF Dump)

ExifTool by Phil Harvey is the most comprehensive command-line tool for reading all EXIF data including MakerNote fields. Run exiftool -all filename.cr3 to dump every tag. The shutter count will appear as ShutterCount or similar depending on brand.

Using Adobe Lightroom / Bridge

Lightroom shows basic EXIF data (exposure, lens, date) in the Metadata panel, but does not expose MakerNote shutter count fields. It is not suitable for verifying shutter count.

Why EXIF Metadata Matters When Buying a Used Camera

EXIF data from an original RAW file provides the most reliable source of objective information about a camera's history. Here's what to check:

  • Shutter count — Total mechanical actuations, compared against the rated shutter lifespan.
  • DateTimeOriginal — When the camera was last used. A camera claimed to be "barely used" with timestamps from three years of regular shooting tells a different story.
  • Camera model — Confirms the file genuinely came from the camera being sold, not a different body.
  • GPS data — If location data is present, it can indicate usage history and region.
Warning: EXIF metadata in image files can be edited using widely available software. The internal hardware counter inside the camera cannot be changed by editing files. Only the file's copy of the metadata is affected. Always verify with an unedited original RAW file for trustworthy results.

Frequently Asked Questions

Does every RAW file format store EXIF the same way?

No. RAW formats — CR3, NEF, ARW, RAF, RW2, ORF, DNG, and others — each have different internal structures. However, all embed EXIF metadata as part of their file format. The standard EXIF fields are consistent, but the MakerNote format varies by brand and sometimes by model generation.

Does converting a RAW to DNG preserve the shutter count?

Adobe DNG conversion preserves the original MakerNote in an embedded sub-block, so the data is technically retained. However, some tools do not read shutter count from DNG wrappers as reliably as from native formats. For verification purposes, use the original native RAW file (CR3, NEF, ARW, etc.).

Why does ExifTool show a different shutter count than ShutterCount.app?

Rarely, there can be differences if a camera stores multiple counters (e.g., total exposures including electronic shutter, vs. mechanical-only actuations). ShutterCount.app extracts the mechanical shutter counter specifically. If you see a discrepancy, the mechanical-shutter counter is the relevant number for assessing shutter wear.

Is GPS location always stored in RAW files?

Only if the camera has GPS capability and location services are enabled. Most dedicated cameras (Canon, Nikon, Sony, Fujifilm) do not have built-in GPS — the GPS fields would only be present if a GPS accessory was attached, or if geotagging was applied in post. Smartphones always embed GPS unless location access is denied.
