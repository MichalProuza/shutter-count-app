શટર કાઉન્ટ પ્રમાણે કૅમેરાની grading

કૅમેરાની rated shutter life સામે count જોઈ સ્થિતિ જાણો:

Gradeઉપયોગ (Rated life ના %)સ્થિતિ
A0–30%ઉત્તમ — લગભગ નવો
B30–60%સારો — ઘણો બાકી
C60–80%ઠીક — ભાવ જોઈ ખરીદો
D80%+જોખમ — shutter જલ્દી બદલવો પડી શકે
ઉદાહરણ: Canon EOS R5 ની rated shutter life 5,00,000 છે. જો શટર કાઉન્ટ 1,50,000 હોય, તો આ Grade A છે.

વાપરેલો કૅમેરો ખરીદવાની સંપૂર્ણ checklist

  • ShutterCount થી શટર કાઉન્ટ ચેક કરો (વેચનાર પાસેથી RAW file માગો)
  • સફ઼ેદ background પર photo લઈ sensor ના dead/hot pixels ચેક કરો
  • Body પર ઊંડા નિશાન, તૂટેલા buttons, અથવા ખરાબ grip ચેક કરો
  • Lens mount માં scratches અથવા ઢીલાપણું ચેક કરો
  • Viewfinder અને LCD screen પર કોઈ ડાઘ તો નહીં
  • બધા buttons, dials અને card slots ચાલે છે
  • Battery charge થાય છે અને કેટલો સમય ચાલે છે
  • Original charger, strap અને box છે કે નહીં

વેચનાર પાસેથી શટર કાઉન્ટ કેવી રીતે માગવો?

વેચનારને કહો કે કૅમેરાથી એક RAW file તમને મોકલે (WhatsApp, email, અથવા Google Drive થી). પછી એ file ShutterCount પર ચેક કરો.

જો વેચનાર RAW file આપવાનો ઇનકાર કરે, તો આ ચેતવણી હોઈ શકે. જે માલિક પ્રામાણિક છે, તે file ચોક્કસ આપશે.

વાપરેલો કૅમેરો ખરીદવા માટે સારી જગ્યાઓ

  • OLX — ભારતનો સૌથી મોટો second-hand marketplace
  • Facebook Marketplace — Local buyers-sellers
  • Flipkart / Amazon (Renewed) — Certified refurbished
  • Local camera shop — જ્યાં તમે કૅમેરો test કરી શકો
