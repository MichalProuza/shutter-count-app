3 ಸರಳ ಹಂತಗಳಲ್ಲಿ ಶಟರ್ ಕೌಂಟ್ ಪರಿಶೀಲಿಸಿ

1
ಕ್ಯಾಮೆರಾದಿಂದ ಫೋಟೋ ತೆಗೆಯಿರಿ ನಿಮ್ಮ ಕ್ಯಾಮೆರಾದಿಂದ ಒಂದು RAW file (Canon CR3/CR2, Nikon NEF, Sony ARW, Fujifilm RAF) ಅಥವಾ JPEG ಫೋಟೋ ತೆಗೆಯಿರಿ. ಈ file ಯಾವ ಕ್ಯಾಮೆರಾದ ಶಟರ್ ಕೌಂಟ್ ತಿಳಿಯಬೇಕೋ ಅದರಿಂದ ತೆಗೆದಿರಬೇಕು.
2
ShutterCount.app ತೆರೆಯಿರಿ ನಿಮ್ಮ browser ನಲ್ಲಿ shuttercount.app ತೆರೆಯಿರಿ. Mobile ಮತ್ತು computer ಎರಡರಲ್ಲೂ ಕೆಲಸ ಮಾಡುತ್ತದೆ. ಯಾವುದೇ app download ಅಥವಾ account ರಚಿಸುವ ಅಗತ್ಯವಿಲ್ಲ.
3
File drag-and-drop ಮಾಡಿ ಫೋಟೋ file ಅನ್ನು webpage ಮೇಲೆ ಎಳೆದು ಬಿಡಿ. ಶಟರ್ ಕೌಂಟ್, ಕ್ಯಾಮೆರಾ model, ಮತ್ತು ಫೋಟೋ ದಿನಾಂಕ ತಕ್ಷಣ ಕಾಣಿಸುತ್ತದೆ.
Privacy guarantee: ನಿಮ್ಮ file ಎಂದಿಗೂ server ಗೆ ಹೋಗುವುದಿಲ್ಲ. ಎಲ್ಲವೂ ನಿಮ್ಮ device ನಲ್ಲೇ ನಡೆಯುತ್ತದೆ.

ಯಾವ ಕ್ಯಾಮೆರಾಗಳು supported?

  • Canon — CR3 (R-series mirrorless), CR2 (DSLR)
  • Nikon — NEF (ಎಲ್ಲಾ Z ಮತ್ತು D-series)
  • Sony — ARW (A7, A6000, ZV-E series)
  • Fujifilm — RAF (X-series, GFX)
  • Olympus / OM System — ORF
  • Panasonic — RW2 (Lumix G/S series)
  • Adobe DNG — Sigma, smartphone
  • JPEG — EXIF MakerNote ಲಭ್ಯವಿದ್ದಾಗ
ಗಮನಿಸಿ: WhatsApp, Instagram ಅಥವಾ Facebook ಮೂಲಕ ಕಳಿಸಿದ ಫೋಟೋಗಳಲ್ಲಿ EXIF data ತೆಗೆದುಹಾಕಲಾಗಿರುತ್ತದೆ. ಯಾವಾಗಲೂ ಕ್ಯಾಮೆರಾ ಅಥವಾ memory card ಯಿಂದ ನೇರ original file ಬಳಸಿ.
