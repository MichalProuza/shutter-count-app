중고 카메라 구매 시 체크리스트

  • 셔터 횟수 확인 — 반드시 실제 사진 파일로 ShutterCount에서 확인합니다. 판매자 제공 스크린샷은 신뢰하지 마세요.
  • 외관 상태 확인 — 바디 긁힘, 마운트 부식, LCD 상태를 직접 확인합니다.
  • 센서 먼지 확인 — 밝은 하늘을 향해 조리개를 좁혀 촬영해 센서 먼지를 확인합니다.
  • 셔터 작동음 확인 — 셔터 작동음이 균일하고 이상음이 없는지 확인합니다.
  • 부속품 확인 — 배터리, 충전기, 스트랩 등 부속품 포함 여부를 확인합니다.
팁: 판매자에게 직접 촬영한 RAW 또는 JPEG 파일을 받아 ShutterCount로 직접 확인하는 것이 가장 정확합니다.
