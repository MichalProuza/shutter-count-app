How to Use It — 3 Steps

  1. Take a photo in RAW mode using the mechanical or EFCS shutter (not silent mode). Most cameras label this as M, Av, Tv, or S mode — just make sure RAW saving is enabled in the menu.
  2. Transfer the RAW file to your computer or phone. Supported formats: CR3, CR2 (Canon), NEF (Nikon), ARW (Sony), RAF (Fujifilm), ORF (Olympus/OM System), RW2 (Panasonic), DNG.
  3. Open shuttercount.app and drop the file onto the drop zone. The shutter count and percentage of rated life consumed appear instantly.

How It Works — Technical Overview

Every RAW file your camera produces contains not just image data, but also a rich block of metadata called EXIF. Inside EXIF is a manufacturer-specific section called the MakerNote, which stores camera settings, firmware version, and — crucially — the shutter actuation count.

ShutterCount's parser:

  • Reads the raw bytes of the file using the browser's FileReader API — no network request is made.
  • Parses the file container (ISOBMFF for CR3, TIFF for NEF/ARW/ORF/RW2, proprietary RAF structure).
  • Locates the EXIF block and navigates to the manufacturer's MakerNote.
  • Reads the shutter count value at the model-specific byte offset.
  • Displays the count alongside the manufacturer's rated shutter life for that model.

The entire process runs in your browser's JavaScript engine. No data is transmitted anywhere.

Frequently Asked Questions

Is ShutterCount really free? No limits?

Yes, completely free. No account, no daily limits, no file size caps (beyond what your browser can handle), no premium tier.

Do I need to install anything?

No. ShutterCount runs in any modern browser — Chrome, Firefox, Safari, Edge. No extensions, no plugins, no app download.

Are my RAW files safe?

Your files never leave your device. There is no server upload. The JavaScript parser reads the file locally using the browser's File API. We have no access to your images.

Can I check shutter count on my phone?

Yes. Open shuttercount.app in your phone's browser and use the file picker to select a RAW file stored on your phone or accessible via cloud storage (Files app on iOS, Files app on Android).

What if my camera isn't supported?

Try it anyway — many models work even if not explicitly listed. If you get no result, the MakerNote structure for that model hasn't been mapped yet. You can contact us with your camera model and we'll look into adding support.
