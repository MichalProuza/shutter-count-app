Co jsou data EXIF a jak obsahují počet expozic?

EXIF (Exchangeable Image File Format) jsou metadata automaticky ukládaná do každého fotografického souboru fotoaparátem. Obsahují informace jako čas expozice, clona, ISO, model fotoaparátu — a v sekci MakerNote také počet expozic závěrky.

Sekce MakerNote je specifická pro každého výrobce. ShutterCount podporuje formáty MakerNote všech hlavních výrobců: Canon, Nikon, Sony, Fujifilm, Olympus a Panasonic.

Soukromí: ShutterCount analyzuje data EXIF výhradně lokálně — soubor nikdy neopustí vaše zařízení.

Jak přečíst EXIF a počet expozic zdarma?

  1. Otevřete ShutterCount — nástroj funguje přímo v prohlížeči, bez instalace.
  2. Načtěte soubor — přetáhněte soubor JPEG nebo RAW (CR3, CR2, NEF, ARW, RAF, ORF, RW2) na stránku nebo klikněte pro výběr souboru.
  3. Přečtěte výsledek — počet expozic a další data EXIF se zobrazí okamžitě.

Podporované formáty souborů

  • Canon — CR3 (bezzrcadlovky R), CR2 (starší zrcadlovky)
  • Nikon — NEF
  • Sony — ARW
  • Fujifilm — RAF
  • Olympus / OM System — ORF
  • Panasonic — RW2
  • JPEG — s originální sekcí EXIF
Upozornění: Fotografie odeslané přes WhatsApp, Instagram nebo Facebook přicházejí o data EXIF, včetně počtu expozic. Vždy používejte originální soubor.

Proč je ShutterCount zdarma?

ShutterCount je open-source projekt udržovaný nadšenci do fotografie. Veškerá analýza probíhá v prohlížeči — nejsou žádné serverové náklady, proto může být nástroj zdarma. Pokud chcete projekt podpořit, můžete využít možnost “Koupit kávu” na hlavní stránce.
