EXIF માં Shutter Count ક્યાં હોય છે?

Shutter Count EXIF data ના MakerNote section માં હોય છે. આ દરેક manufacturer માટે અલગ જગ્યા અને અલગ format માં હોય છે:

BrandFormatEXIF Location
CanonCR3, CR2CTMD block (CR3) / MakerNote IFD (CR2)
NikonNEFMakerNote → ShutterCount tag
SonyARWMakerNote → ImageCount
FujifilmRAFMakerNote → FrameNumber
Olympus/OMORFMakerNote → Equipment IFD
PanasonicRW2MakerNote → ShutterCount

ShutterCount EXIF કેવી રીતે વાંચે છે?

ShutterCount સંપૂર્ણ તમારા browser માં JavaScript થી ચાલે છે. File ને JavaScript ના FileReader API થી વાંચવામાં આવે છે અને MakerNote bytes ને સીધા parse કરવામાં આવે છે.

Privacy: તમારી photo file નો એક પણ byte server પર જતો નથી. બધું તમારા device પર થાય છે.

JPEG file માંથી પણ Shutter Count વાંચી શકાય?

હા, પણ દરેક કૅમેરો JPEG માં shutter count store નથી કરતો. Canon R-series, Nikon Z-series અને Sony A-series ના JPEG માં સામાન્ય રીતે shutter count હોય છે. RAW files માં વધુ વિશ્વસનીય હોય છે.

ExifTool command line પર કેવી રીતે ચેક કરવો?

જો તમે technical છો અને ExifTool use કરવા ઈચ્છો:

exiftool -ShutterCount photo.NEF
exiftool -ImageCount photo.ARW
exiftool -ShutterCount photo.CR2

ExifTool મફત અને open-source છે, પણ તેના માટે command line ની જાણ જોઈએ. સરળ રીત માટે ShutterCount.app વધુ સારો છે.
