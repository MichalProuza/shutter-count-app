ಶಟರ್ ಕೌಂಟ್ ಪ್ರಕಾರ ಕ್ಯಾಮೆರಾ grading

ಕ್ಯಾಮೆರಾದ rated shutter life ಎದುರು count ನೋಡಿ ಸ್ಥಿತಿ ತಿಳಿಯಿರಿ:

Gradeಬಳಕೆ (Rated life ನ %)ಸ್ಥಿತಿ
A0–30%ಅತ್ಯುತ್ತಮ — ಬಹುತೇಕ ಹೊಸತು
B30–60%ಉತ್ತಮ — ಬಹಳ ಬಾಕಿ ಇದೆ
C60–80%ಸಾಧಾರಣ — ಬೆಲೆ ನೋಡಿ ಖರೀದಿಸಿ
D80%+ಅಪಾಯ — shutter ಶೀಘ್ರದಲ್ಲೇ ಬದಲಾಯಿಸಬೇಕಾಗಬಹುದು
ಉದಾಹರಣೆ: Canon EOS R5 ನ rated shutter life 5,00,000. ಶಟರ್ ಕೌಂಟ್ 1,50,000 ಇದ್ದರೆ ಇದು Grade A.

ಹಳೆಯ ಕ್ಯಾಮೆರಾ ಖರೀದಿಸುವ ಸಂಪೂರ್ಣ checklist

  • ShutterCount ನಿಂದ ಶಟರ್ ಕೌಂಟ್ ಪರಿಶೀಲಿಸಿ (ವಿಕ್ರೇತರಿಂದ RAW file ಕೇಳಿ)
  • ಬಿಳಿ background ನಲ್ಲಿ ಫೋಟೋ ತೆಗೆದು sensor ನ dead/hot pixels ಪರಿಶೀಲಿಸಿ
  • Body ಮೇಲೆ ಆಳವಾದ ಗೀರುಗಳು, ಮುರಿದ buttons, ಅಥವಾ ಕೆಟ್ಟ grip ಪರಿಶೀಲಿಸಿ
  • Lens mount ನಲ್ಲಿ scratches ಅಥವಾ ಸಡಿಲ ಇದೆಯೇ ನೋಡಿ
  • Viewfinder ಮತ್ತು LCD screen ಮೇಲೆ ಕಲೆಗಳಿಲ್ಲ ಎಂದು ಖಾತ್ರಿ ಮಾಡಿ
  • ಎಲ್ಲಾ buttons, dials ಮತ್ತು card slots ಕೆಲಸ ಮಾಡುತ್ತವೆ
  • Battery charge ಆಗುತ್ತದೆ ಮತ್ತು ಎಷ್ಟು ಸಮಯ ಉಳಿಯುತ್ತದೆ
  • Original charger, strap ಮತ್ತು box ಇದೆಯೇ

ವಿಕ್ರೇತರಿಂದ ಶಟರ್ ಕೌಂಟ್ ಹೇಗೆ ಕೇಳುವುದು?

ವಿಕ್ರೇತರನ್ನು ಕ್ಯಾಮೆರಾದಿಂದ ಒಂದು RAW file ಕಳಿಸಲು ಹೇಳಿ (WhatsApp, email, ಅಥವಾ Google Drive ಮೂಲಕ). ನಂತರ ಆ file ಅನ್ನು ShutterCount ನಲ್ಲಿ ಪರಿಶೀಲಿಸಿ.

ವಿಕ್ರೇತರು RAW file ಕೊಡಲು ನಿರಾಕರಿಸಿದರೆ, ಇದು ಎಚ್ಚರಿಕೆಯ ಸಂಕೇತ. ಪ್ರಾಮಾಣಿಕ ಮಾಲೀಕರು ಖಂಡಿತ file ಕೊಡುತ್ತಾರೆ.

  • OLX — ಭಾರತದ ಅತಿದೊಡ್ಡ second-hand marketplace
  • Facebook Marketplace — ಸ್ಥಳೀಯ buyers-sellers
  • Flipkart / Amazon (Renewed) — Certified refurbished
  • ಸ್ಥಳೀಯ camera shop — ಅಲ್ಲಿ ನೀವು ಕ್ಯಾಮೆರಾ test ಮಾಡಬಹುದು
