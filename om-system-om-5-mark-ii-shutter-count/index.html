How to Check Shutter Count on the OM System OM-5 Mark II

The OM System OM-5 Mark II uses ORF (Olympus RAW File) format. Unlike Canon CR3 or Sony ARW files, ORF shutter count embedding is not confirmed for all firmware versions. Two methods are available:

Method 1: Camera menu (recommended)

  1. Power on the OM-5 Mark II.
  2. Press MENU and navigate to the Setup Menu (wrench icon).
  3. Scroll to find the shutter count display (location varies by firmware — check the last page of the Setup menu if not immediately visible).
  4. The total mechanical shutter count is displayed on screen.

Method 2: ORF file via ShutterCount

  1. Take a photo in RAW mode and locate the resulting .ORF file.
  2. Open shuttercount.app in any modern browser.
  3. Drag the ORF file onto the drop zone.
  4. If the shutter count is embedded in the ORF MakerNote for your firmware version, it will be displayed. If not, the camera menu method is required.
ORF shutter count caveat: OM System (Olympus) ORF files do not reliably embed the shutter count in a confirmed standard tag. ShutterCount will attempt to read available MakerNote fields, but the camera menu method is the most reliable approach for OM-5 Mark II bodies.

OM System OM-5 Mark II vs OM-5 — What Changed?

The OM-5 Mark II (2024) builds on the original OM-5 (2022) with these key improvements:

  • AI subject recognition AF: Expanded to cover birds, insects, vehicles, and more
  • 4K/60p video: Up from 4K/30p on the original OM-5
  • OM-Log profile: Updated OM-Log for improved video colour science
  • Improved IBIS: Up to 7.5-stop stabilisation (vs 6.5-stop on OM-5)
  • Handheld High-Res Shot: 50 MP composite retained

The sensor (20 MP MFT BSI CMOS), weather sealing (IP53), body dimensions, and shutter mechanism are essentially unchanged from the original OM-5.

OM System OM-5 Mark II Shutter Count — FAQ

How do I check the shutter count?

The most reliable method is via Menu → Setup → shutter count display on the camera. You can also drop an ORF RAW file into shuttercount.app — the tool will attempt to extract the count from EXIF MakerNote data if embedded by your firmware.

What is the rated shutter life?

OM System does not publish an official rating. The estimated lifespan is approximately 150,000 actuations.

Does the electronic shutter affect the count?

No. The OM-5 Mark II supports a fully electronic shutter mode. Electronic shutter exposures do not actuate the mechanical curtain and do not increment the mechanical shutter count. The Handheld High-Res Shot mode (which composites multiple frames) uses the mechanical shutter for each sub-frame — be aware that this mode increments the count more rapidly.
