什么是 EXIF 数据？

EXIF（Exchangeable Image File Format，可交换图像文件格式）是嵌入在照片文件中的元数据标准，包含拍摄时间、ISO、光圈、快门速度、相机型号等信息。

对于快门次数，相机会将当前的快门累计次数写入每张照片的 EXIF 中，字段通常为 ShutterCount、ImageNumber 或品牌私有 Makernote 字段。ShutterCount 能识别 60 多种相机型号的快门次数 EXIF 位置。

隐私说明：所有文件处理完全在你的浏览器中完成，不会上传至任何服务器，你的照片数据始终在本地。

如何用 ShutterCount 读取 EXIF 快门数据？

  1. 准备文件 — 用相机拍一张照片，保存为 JPEG 或 RAW 格式（CR3、NEF、ARW 等）。注意：手机拍摄的照片不含快门次数。
  2. 打开 ShutterCount — 访问 shuttercount.app，将文件拖入页面，或点击选择文件。
  3. 即时解析 — 工具在浏览器本地读取 EXIF 数据，快门次数、机型及快门健康度瞬间呈现。
  4. 批量查询 — 支持一次拖入多个文件，自动批量解析，结果可导出为 CSV。
