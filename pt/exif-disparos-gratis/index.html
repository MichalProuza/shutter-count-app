Cada fotografia digital contém metadados invisíveis chamados EXIF (Exchangeable Image File Format). Entre esses dados está gravado o número total de disparos da câmera. O ShutterCount lê esses dados automaticamente — sem software, sem conta, sem pagar nada.

O que são os dados EXIF?

Os dados EXIF são informações técnicas gravadas automaticamente em cada fotografia pela câmera. Incluem:

  • Data e hora em que a fotografia foi tirada
  • Definições de exposição (abertura, velocidade, ISO)
  • Modelo e número de série da câmera
  • Número de disparos do obturador (campo MakerNote, específico de cada fabricante)

O contador de disparos está guardado numa secção privada dos metadados EXIF chamada MakerNote — uma área proprietária de cada fabricante. O ShutterCount sabe como ler essa secção para os principais fabricantes.

Como ver os dados EXIF do contador de disparos — passo a passo

  1. Tire uma fotografia com a câmera que quer verificar.
  2. Transfira a fotografia para o computador ou telemóvel — não use fotografias enviadas por WhatsApp ou redes sociais: estas plataformas removem os metadados EXIF.
  3. Carregue o ficheiro JPEG ou RAW no ShutterCount.
  4. O contador de disparos aparece de imediato.
Atenção: Fotografias partilhadas via WhatsApp, Instagram ou Facebook têm os dados EXIF removidos automaticamente. Carregue sempre o ficheiro original diretamente da câmera ou do cartão de memória.

Alternativas gratuitas para ler EXIF

Além do ShutterCount, existem outras formas de ler os dados EXIF:

  • ExifTool — software gratuito para linha de comandos, muito completo mas técnico.
  • Jeffrey's Exif Viewer — visualizador online de EXIF genérico.
  • Opanda IExif — extensão para Windows Explorer.

Nenhuma destas alternativas apresenta o contador de disparos de forma tão direta e simples quanto o ShutterCount.
