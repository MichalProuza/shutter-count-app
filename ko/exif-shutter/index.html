EXIF 데이터란 무엇인가요?

EXIF(Exchangeable Image File Format)는 디지털 카메라가 사진 파일에 자동으로 저장하는 메타데이터 표준입니다. EXIF에는 다음과 같은 정보가 포함됩니다.

  • 촬영 날짜 및 시간
  • 카메라 모델 및 제조사
  • 셔터 속도, 조리개, ISO
  • GPS 위치 정보 (지원 기종에 한함)
  • 셔터 횟수 (MakerNote 섹션에 저장, 제조사마다 방식이 다름)

셔터 횟수는 EXIF 어디에 저장되나요?

셔터 횟수는 표준 EXIF 태그가 아니라 각 제조사 고유의 MakerNote 섹션에 저장됩니다. 위치와 형식이 제조사별로 다르기 때문에 일반적인 EXIF 뷰어로는 확인이 어렵습니다.

ShutterCount가 표시하는 EXIF 정보

셔터 횟수 외에도 ShutterCount는 다음 EXIF 정보를 표시합니다.

  • 카메라 제조사 및 모델명
  • 파일 형식 (CR3, NEF, ARW 등)
  • 촬영 날짜
  • 셔터 수명 대비 사용률 시각화 (라이프바)
