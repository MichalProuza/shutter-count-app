Cum citești contorul de declanșări?

  1. Fotografiază orice subiect — cu aparatul al cărui număr de declanșări vrei să îl afli.
  2. Încarcă fișierul — trage fișierul JPEG sau RAW direct pe această pagină.
  3. Citește rezultatul — contorul de declanșări este afișat instant din datele EXIF.
Confidențialitate: Fotografia ta nu este niciodată trimisă pe server. Analiza se face complet în browser — 100 % privat.

De ce să verifici contorul de declanșări?

Contorul de declanșări indică uzura reală a obturatorului mecanic. Este esențial atunci când:

  • Vrei să cumperi un aparat foto second hand și să îi cunoști starea reală.
  • Dorești să vinzi aparatul tău și să oferi informații corecte cumpărătorului.
  • Vrei pur și simplu să știi câte fotografii a făcut aparatul tău până acum.

Ce este contorul de declanșări?

Contorul de declanșări (cunoscut și ca shutter count în engleză) este un numărător intern care se incrementează la fiecare declanșare. Acesta este stocat în secțiunea MakerNote a datelor EXIF din fiecare fotografie și nu poate fi resetat cu ușurință.

Acest contor este comparabil cu kilometrajul unei mașini second hand: cu cât este mai mare, cu atât obturatorul este mai uzat.

Ce aparate sunt suportate?

ShutterCount suportă majoritatea reflexelor și aparatelor mirrorless ale marilor mărci: Canon (CR3, CR2), Nikon (NEF), Sony (ARW), Fujifilm (RAF), precum și Olympus și Panasonic. Fișierele JPEG sunt de asemenea suportate.
